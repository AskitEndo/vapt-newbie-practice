## ğŸ§° Learning Path

<p align="center">

This lab series follows a logical progression from passive reconnaissance to active exploitation:

1. **Information Gathering** (Labs 01-03): Learn non-intrusive techniques to collect target information
2. **Vulnerability Assessment** (Lab 04): Identify potential security weaknesses systematically
3. **Exploitation** (Labs 05A-06): Practice actual exploitation techniques in safe environments

Each lab builds upon skills from previous exercises while introducing new concepts and tools.
